## Getting Started

tools

- compiler: `nasm`
- VM: `dosbox`

````bash
nasm $source_file -fbin -o ${out}.com

# start dosbox in a separate terminal
dosbox

```dos
mount c ~/$source_directory

REM go to C disk
c:

REM list files
dir

REM execute compiled binary
${out}.com
````
